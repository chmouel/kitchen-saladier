POST /versions
==============

Associate a product to a version.

Arguments::

    {
        "url": "http://somewhereoutthere",
        "version": "1.0",
        "product_id": "d6c308c9-20bd-4160-bf62-041b7031187c"
    }

Returns::

    201

.. note:: This call needs to be made with the admin rights.
